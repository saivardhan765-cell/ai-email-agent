AI Email Assistant

What it's?:

This project is an AI-powered email assistant that:

- Classifies emails
- Assigns priority
- Generates replies
- Evaluates performance using a reward system

🚀 Features

- Email classification (Work, Personal, Spam)
- Priority detection (High, Medium, Low)
- Auto reply generation
- Reward-based evaluation system
- Interactive UI with animation

🛠️ Tech Stack

- Python
- Streamlit

▶️ How to Run

streamlit run app.py

📊 Output

- Classification
- Priority
- Suggested Reply
- Reward Score

🎯 Purpose

Built for hackathon/demo to simulate intelligent email handling using AI logic.
